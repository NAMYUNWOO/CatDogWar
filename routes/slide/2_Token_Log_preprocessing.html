본 Script 에서는, Token 발신 Log를 활용하여 정형변수를 추출한다.
Token 발신은 각 직원이 다른 직원에게 감사의 의미로 사내 기프티콘과 같은 선물을 보내는 것을 말한다

  1. 로그 수집기간: 2016.06 ~ 2018.06
  2. 정형화1: 각 사원마다 월별 Token발신, 수신 횟수를 각각 전체 월에대하여 평균을 냄
  3. 정형화2: 정형화1의값을 그룹 직급별로 평균내어, 각 사원마다 직급평균 대비 value (ratio)를 구함
  4. 결과: 총 4개 변수를 추출함
    1) Token send count monthly average
    2) Token receive count monthly average
    3) Token send count monthly average GroupBy Level ratio
    4) Token receive count monthly average GroupBy Level ratio

def mean_except_outlier(series):
    q1 = series.quantile(0.25)
    q3 = series.quantile(0.75)
    iqr = q3 - q1 
    series2 = series[series > q1 - N_IQR*iqr]
    if len(series2) == 0:
        return np.mean(series)
    return np.mean(series2)
